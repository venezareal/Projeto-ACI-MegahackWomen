* COMO ESSE MODELO VAI FUNCIONAR?



A grosso modo, um modelo de Machine Learning funciona através de um conjunto de cálculos feitos com base em um conjunto de dados que busca encontrar um padrão nesse conjunto de dados para assim conseguir prever o se que deseja.

Por exemplo, no caso da análise de crédito, para fins didáticos, digamos que usemos um dataset com o cadastro de 1000 clientes que contém o nome, email, dados pessoais, endereço, número do celular, score de crédito e situação no SPC ou Serasa.

Digamos que queremos que o modelo análise esses dados e consiga classificar um cliente como bom pagador ou mau pagador. Aqui estaremos usado o método de aprendizado supervisionado, onde dizemos o que significa cada variável e qual resultado queremos que modelo entregue. Esse dataset é separado em dois conjuntos de dados, um para treino e outro para teste. E o modelo é refinado até ele atingir a margem de erro desejada.



* QUAIS DADOS ESTAMOS UTILIZANDO?



O dataset foi feito com base em relatórios do SERASA sobre o perfil do consumidor inadimplente e adimplentes. Com a autorização dos usuários, estamos utilizando um questionário socioeconômico básico para capturar os dados relevantes que serão tratados e inseridos no nosso modelo. Posteriormente esse questionário será adaptado para uma forma de game financeiro, e disponibilizado em plataformas sociais como facebook e instagram. Permitindo assim capturar com maior precisão o contexto de consumo da pessoa.



* O QUE ESTAMOS RETORNANDO COM ESSE MODELO?



Após processar os dados capturados do formulário, nosso modelo retornará para as fintechs financeiras um arquivo contendo as informações relevantes sobre um grupo de pessoas, quanto a essas pessoas estarem aptas para receber crédito ou não.
